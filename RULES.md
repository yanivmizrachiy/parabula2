# RULES.md — שער כניסה קצר

מקורות אמת:
- `PROJECT_RULES.md` — החוזה המלא
- `rules.html` — גרסה קריאה

## חובה לפני כל שינוי
- להריץ `npm run preview`
- לוודא שאין inline CSS
- לוודא שאין inline JS
- לוודא שהתצוגה מתחילה מראש הדף
- לוודא שהעמוד ממורכז
- לוודא שאין מסך ריק ב-`/preview`

## חובה לפני פרסום
- להריץ `npm test`
- להריץ build ל-`site/`
- לוודא ש-`site/` מכיל בדיוק את דפי הספר/העבודה הצפויים לפני push ל-main
